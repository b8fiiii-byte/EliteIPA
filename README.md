# Elite IPA — تطبيق توقيع وتثبيت IPA مباشر (بديل ESign)

تطبيق iOS احترافي لتوقيع تطبيقات الـ IPA باستخدام شهادتك (.p12 + .mobileprovision) وتثبيتها **مباشرة من داخل التطبيق** دون الحاجة لحاسوب، بواجهة فخمة (Dark Mode + Glassmorphism). يعمل على الأجهزة العادية (عبر الشهادة) ويدعم أجهزة الجيلبريك (AppSync) أيضاً.

## المميزات
- **مدير ملفات متقدم:** استعراض، فك ضغط، حذف، إعادة تسمية، إضافة، ومعاينة.
- **إدارة الشهادات:** استرداد `.p12` و `.mobileprovision`، حفظ كلمة السر في Keychain، وفحص الصلاحية (صالحة/مقفلة/منتهية) عبر OCSP.
- **محرك توقيع zsign:** تعديل الاسم، الأيقونة، الـ Bundle ID، وحقن مكتبات dylib.
- **التثبيت المباشر:** يتم بالكامل داخل التطبيق عبر سيرفر HTTPS محلي + `itms-services` (وAppSync على الجيلبريك).
- **مكتبة التطبيقات:** عرض التطبيقات المستوردة بأيقوناتها وتفاصيلها.
- **تصدير الـ IPA الموقّع** ومشاركته.
- **قناة تيليجرام:** https://t.me/EliteIPA

## كيف تحصل على ملف الـ IPA (بدون امتلاك ماك)

هذا المشروع مكتوب بـ Swift/SwiftUI، وبناء أي IPA يتطلب أدوات Apple على ماك. الحل: **GitHub Actions** يوفّر ماك سحابي مجاني يبني لك الـ IPA تلقائياً.

### الخطوات
1. أنشئ مستودعاً جديداً على GitHub وارفع محتويات هذا المجلد إليه.
2. اذهب إلى تبويب **Actions** في المستودع، وفعّل الـ Workflows.
3. شغّل Workflow باسم **"Build Elite IPA"** (يدوياً عبر زر *Run workflow* أو تلقائياً عند الرفع).
4. بعد انتهاء البناء (بضع دقائق)، نزّل ملف **EliteIPA.ipa** من قسم *Artifacts* في صفحة التشغيل.

> الـ IPA الناتج **غير موقّع** (Unsigned)، وهذا متوقّع: التطبيق نفسه يُوقّع على جهازك بشهادتك مثل أي تطبيق ESign. وقّعه بأداتك المفضّلة (Sideloadly / AltStore / أو حتى Elite IPA نفسه بعد تثبيته أول مرة) ثم ثبّته.

## بنية المشروع
- `OFFSign/` — كود التطبيق (Swift/SwiftUI).
  - `Views/` — الشاشات (الملفات، المكتبة، التفاصيل، التوقيع، الإعدادات).
  - `ViewModels/` — منطق الواجهات.
  - `Services/` — التخزين، Keychain، التوقيع، الفحص، التثبيت، السيرفر المحلي.
  - `Engine/` — جسر zsign (C++).
  - `Resources/` — Info.plist، entitlements، الثيم.
- `project.yml` — توليد مشروع Xcode عبر XcodeGen.
- `.github/workflows/build-ipa.yml` — بناء الـ IPA تلقائياً.

## ملاحظات تقنية
- التوقيع والتثبيت يعتمدان **كلياً على شهادتك**. أضف الشهادة من الإعدادات > استرداد الشهادة، أدخل كلمة السر، فعّلها، ثم وقّع وثبّت أي IPA.
- على الأجهزة بدون جيلبريك: لا بد أن تكون الشهادة تغطي الجهاز (UDID في الـ provision أو شهادة Enterprise) ليعمل التثبيت عبر `itms-services`.
- على الجيلبريك: يستخدم التطبيق AppSync للتثبيت الفوري.
